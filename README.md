# Workshop on Interactive AI agent

Welcome to our workshop on Interactive AI Agent! Following this workshop, you will be able to experience

* Establishing your local large lanaguage model (LLM)(s)
* Creating an Angentic Application through 
* Working with your local LLM in the AI agentic application 
* Building your own interactive agentic AI by learning the behind-the-scene work!

To do so, here are the tutorials:

* Tutorial 0: Setting up your computer
* Tutorial Example: Running your 1st local Agentic AI application!
* Tutorial 1: Working with the basics - Ollama to run an LLM on your computer
* Tutorial 2: Working with the basics - Python for the instructions
* Tutorial 3: Working with the intermediates 1 - Streamlit for the interface
* Tutorial 4: Working with the intermediates 2 - Pocketflow for the AI agents
* Tutorial Bonus: Finetuning your LLM for a very specific application

But before all that, you need to download this repository to your own computer. On the top right hand corner, click the button "Code" -> "Download ZIP" and extract all files to begin!
